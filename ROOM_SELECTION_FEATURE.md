# 房间选择功能

## 功能概述

在餐厅预订流程中增加了房间选择步骤，用户可以在选择菜单之前先选择合适的用餐空间。

## 更新内容

### 1. 新增组件：RoomSelection.vue

位置：`src/components/booking/RoomSelection.vue`

功能：

- 展示所有可用房间的卡片列表
- 每个房间卡片包含：
  - 房间照片
  - 房间名称和描述
  - 设施标签（麻将房、独立包间、茶室等）
  - 可容纳人数
- 支持点击选择房间
- 选中的房间会有视觉高亮效果

### 2. 更新预订流程

位置：`src/views/booking/RestaurantBookingView.vue`

变更：

- 步骤从 3 步增加到 4 步：
  1. **选择房间**（新增）
  2. 选择菜单
  3. 选择时间
  4. 填写信息

- 在 bookingData 中增加 `selectedRoom` 字段
- 第一步必须选择房间后才能进入下一步

### 3. 房间类型

目前支持的房间类型：

1. **麻将房**
   - 独立包间，配备全自动麻将桌
   - 可容纳 8 人
   - 设施：麻将桌、独立包间、空调、音响系统

2. **独立包间**
   - 私密舒适的用餐空间
   - 可容纳 12 人
   - 设施：独立包间、空调、电视、独立卫生间

3. **公开位置**
   - 开放式用餐区域
   - 可容纳 20 人
   - 设施：开放空间、自然采光、空调

4. **茶室**
   - 雅致的茶室空间
   - 可容纳 6 人
   - 设施：茶具、独立包间、古典装饰、空调

5. **花园**
   - 户外花园用餐区
   - 可容纳 15 人
   - 设施：户外空间、花园景观、遮阳伞、户外音响

## 使用说明

用户在预订时：

1. 首先看到所有可用房间的展示
2. 根据人数、设施需求选择合适的房间
3. 点击房间卡片进行选择
4. 选择后点击"下一步"继续选择菜单

## 技术细节

- 使用 Vue 3 Composition API
- 使用 `defineModel` 实现双向绑定
- 响应式设计，支持移动端和桌面端
- 使用 Tailwind CSS 进行样式设计
- 支持悬停效果和选中状态的视觉反馈
